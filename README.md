# CS GitHub planning guide (CS1 / CS2)

Course materials for **`planning.md`**: policy, student-facing expectations, a 4-point rubric, and full examples (web + programming).

## Start here

Open **[CS_GITHUB_PLANNING_GUIDE.md](CS_GITHUB_PLANNING_GUIDE.md)** on GitHub for the full guide. In that file, **Jump to sections** links use anchors like `#part-a` (they work on github.com).

## Files

| File | Purpose |
|------|---------|
| [CS_GITHUB_PLANNING_GUIDE.md](CS_GITHUB_PLANNING_GUIDE.md) | Unified instructor + student guide |
| [planning_web.md](planning_web.md) | CS1 portfolio-style example only |
| [planning_codes.md](planning_codes.md) | CS2 max/min/average example only |
| [planing_rubrics.md](planing_rubrics.md) | Rubric only |

## Use with assignments

- **Download:** green **Code** → **Download ZIP**, or clone this repo.
- **Template (optional):** In the GitHub repo settings, enable **Template repository** so students can use **Use this template** for each new assignment.

Students should copy the structure into their own assignment repo as **`planning.md`** and replace the task-specific text with the current assignment.
